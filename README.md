## 🛠️ Tecnologias Utilizadas:

A interface do utilizador foi construída utilizando tecnologias web nativas para garantir a máxima performance, leveza e compatibilidade:

* **HTML5:** Estruturação semântica do conteúdo, otimizada para SEO (motores de pesquisa como o Google).
* **CSS3:** Estilização moderna com design responsivo (adaptado para telemóveis/iPhones e computadores) através de Media Queries.
* **JavaScript (Vanilla):** Lógica do lado do cliente para controlo de animações, manipulação do DOM e envio assíncrono dos dados do formulário através da Fetch API.
* **Vercel:** Plataforma utilizada para o alojamento do site e integração contínua (Deploy automático a cada atualização do código).

---

## 📦 Funcionalidades Implementadas:

* 📱 **Design Mobile-First & Responsivo:** Interface perfeitamente adaptada para ecrãs de smartphones (iOS/Android) e monitores desktop.
* ✍️ **Formulário de Feedback Dinâmico:** Permite aos clientes avaliar os serviços dos barbeiros diretamente na página.
* ⚡ **Validação em Tempo Real:** O botão de envio é desativado temporariamente após o clique para evitar múltiplos envios acidentais de e-mail.
* 🔍 **Otimização SEO:** Configuração de Meta Tags (`viewport`, `description`, `keywords`) validada e indexada no Google Search Console.
